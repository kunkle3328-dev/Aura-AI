# Aura AI: A Conversational AI with a Personality

Aura AI is a conversational AI application that provides a natural, human-like voice and video interaction experience. It is designed to be an engaging and empathetic partner, with a witty personality. Aura can understand and respond to your voice, and it can also see and understand the world through your camera.

## Features

- **Voice and Video Interaction:** Aura can understand and respond to your voice, and it can also see and understand the world through your camera.
- **Affective Dialog:** Aura is designed to be an empathetic and witty partner, with a personality that is engaging and fun.
- **Customizable Avatar:** You can customize the appearance of Aura's avatar to your liking.
- **Extensible with Tools:** Aura can be extended with tools to perform tasks such as getting the weather or setting reminders.

## Getting Started

To get started with Aura AI, you will need to have Node.js and npm installed on your machine. You will also need a Gemini API key.

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)
- [Gemini API Key](https://ai.google.dev/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/aura-ai.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Create a `.env.local` file in the root of the project and add your Gemini API key:
   ```
   GEMINI_API_KEY=your-api-key
   ```
4. Run the application:
   ```bash
   npm run dev
   ```
The application will be available at [http://localhost:3000](http://localhost:3000).

## Usage

Once the application is running, you can interact with Aura by speaking to it. You can also use the controls at the bottom of the screen to start and stop the conversation, as well as toggle the camera.

### Customizing the Avatar

You can customize the appearance of Aura's avatar by clicking on the customize icon in the top right corner of the screen. This will open a panel where you can change the avatar's style, texture, shape, and color.

### Using Tools

Aura can be extended with tools to perform tasks such as getting the weather or setting reminders. To use a tool, simply ask Aura to perform the task. For example, you can say "What's the weather like in San Francisco?" or "Set a reminder to take out the trash in 5 minutes."
